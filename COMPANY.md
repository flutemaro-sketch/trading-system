# トレーディング・カンパニー 会社憲法

## 会社ビジョン

トレードシステムを「会社組織」として機能させる。
各部署が独立して動き、データを共有しながら連携する完全自動監視システムを目指す。

---

## 組織図

### 【第1部署】前日検討部
- **状態:** ✅ 稼働中（初代秀長）
- **機能:** JPXデータ連携、翌日の有望銘柄をAIスコアでスクリーニング
- **出力:** `shared/watchlist.csv`

### 【第2部署】上昇シグナル部
- **状態:** ✅ 実戦好調（2代目秀長）
- **機能:** 板監視 + ペナント上抜けアラート
- **実績:** 2026-05-22 に28件検知・3銘柄ストップ高達成
- **出力:** `shared/signals/rise/`

### 【第3部署】下降シグナル部
- **状態:** 🔧 開発中（2代目秀長）
- **機能:** 下がるパターンの検知・アラート
- **学習中:** `learning-data/PATTERNS.md` 参照
- **出力:** `shared/signals/fall/`

### 【第4部署】上値目標部
- **状態:** 📋 構想中（フェーズ3 / 新PC到着後）
- **機能:** 上昇検知後に目標価格を自動計算・表示
- **計算指標:** 直近高値・VWAP・出来高だまり上限・抵抗線
- **出力:** `shared/targets/`

### 【第5部署】下値目標部
- **状態:** 📋 構想中（フェーズ3 / 新PC到着後）
- **機能:** 下降検知後に下値目標を自動計算・表示
- **計算指標:** 直近安値・支持線・出来高だまり下限
- **出力:** `shared/targets/`

---

## 部署間の連携図

```
【第1部署】前日検討部
    ↓ shared/watchlist.csv（有望銘柄リスト）
【第2部署】上昇シグナル部  ← 現在好調
    ↓ shared/signals/rise/（上昇検知データ）
【第4部署】上値目標部      ← フェーズ3

【第3部署】下降シグナル部  ← 開発中
    ↓ shared/signals/fall/（下降検知データ）
【第5部署】下値目標部      ← フェーズ3
```

---

## 共有データ構成

```
trading-system/shared/
├── watchlist.csv        ← 第1部署が更新（有望銘柄リスト）
├── signals/
│   ├── rise/            ← 第2部署（上昇シグナル）
│   └── fall/            ← 第3部署（下降シグナル）
└── targets/             ← 第4・第5部署（目標価格）
```
