## 現在の仕様

### コア機能
```
kabuStation API (WebSocket)
    ↓
  リアルタイム板データ
    ↓
  Tkinter GUI （拡大禁止）
    ↓
  50銘柄同時監視
```

### UI レイアウト
- 上下2段のタブ（各5タブ）
- 各タブに5銘柄×横並び
- 1銘柄 = 140px幅（expand=False 固定）
- ヘッダ：銘柄コード、現在値、騰落率
- 情報行：始値、高値、安値、出来高
- 板：売5段 + 買5段

### データフロー
```
Queue （データ受信）
  ↓
MainWindow._process_data()
  ↓
BoardPanel.update_data()
  ↓
Tkinter GUI 更新
```

## アーキテクチャノート

### 現在のシンプルさの価値
- lion シグナル計算ロジックがないため、J-Quants 日足データが不要
- DailyContextStore の日付キャッシュ管理が不要
