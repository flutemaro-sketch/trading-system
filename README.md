# 日本株 前日検討システム

## 概要

らいおんまるさんの手法に基づいた、日本株の前日検討スクリーニングシステム。
登録した銘柄リスト(100銘柄)を毎日引け後にスコアリングして、翌日の候補銘柄を抽出する。

詳細仕様は [SPEC.md](./SPEC.md) を参照。

## アーキテクチャ

- **データ取得**: J-Quants API (Light プラン)
- **将来拡張**: kabuステーションAPI (auカブコム証券・寄付き判断モジュール)
- **言語**: Python 3.14 / Windows

## パターン学習システム

スクショを見せながら少しずつパターンを学習し、検知ロジックに落とし込む仕組み。

**学習フロー:**
1. スクショを `learning-data/up-patterns/` または `down-patterns/` に保存
2. `learning-data/PATTERNS.md` にパターン条件を記録
3. 条件が固まったら検知ロジックをコードに実装

**現在実装済みパターン:**
- ▲ ペナント上抜け（`pennant_detector.py`）
- ▲ 始値ブレイク / 始値接近
- ▲ OVER/UNDER逆転（買い圧力増加）

**学習予定パターン:**
- ▼ 始値抵抗で跳ね返し
- ▼ VWAP抵抗で反落
- ▼ 移動平均線全下向き + VWAP下 + 板悪化
- ▼ ペナント下抜け

詳細は [`learning-data/PATTERNS.md`](./learning-data/PATTERNS.md) を参照。
