# 日本株 前日検討システム

## 概要

らいおんまるさんの手法に基づいた、日本株の前日検討スクリーニングシステム。
登録した銘柄リスト(100銘柄)を毎日引け後にスコアリングして、翌日の候補銘柄を抽出する。

詳細仕様は [SPEC.md](./SPEC.md) を参照。

## アーキテクチャ

- **データ取得**: J-Quants API (Light プラン)
- **将来拡張**: kabuステーションAPI (auカブコム証券・寄付き判断モジュール)
- **言語**: Python 3.14 / Windows
