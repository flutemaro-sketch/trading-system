# 証券会社ツール 研究メモ

作成日: 2026-05-14
目的: 板監視システム（board_monitor）v2 のUI設計、および将来のトレーディング画面構築の参考にするため、実在する証券会社ツールの板表示・レイアウト思想を整理する。

## 1. 各証券会社の板表示レイアウト

### 1.1 SBI証券 HYPER SBI 2
- **レイアウト思想**: タイル形式で複数銘柄を分割表示
- **特徴**:
  - 1画面を格子状（タイル）に区切り、各セルに1銘柄の板を配置
  - サイズは固定気味で、銘柄数を増やすと1セルが小さくなる
  - 板の段数は売買とも上位5本程度がデフォルト
- **学べる点**: 「画面占有率」と「視認性」のトレードオフをタイル化で解決している

### 1.2 kabuステーション（auカブコム証券）
- **レイアウト思想**: 複数ウィンドウを自由に配置するレイアウトフリー型
- **特徴**:
  - 各ウィンドウが独立しており、サイズ・位置をユーザーが自在に決められる
  - 板・歩み値・チャート・ニュースなどパーツ単位で配置
  - 当システムが採用している kabuStation API もここの提供
- **学べる点**: 「テンプレートを押しつけない」自由度の高さ。ただし初期設定コストは高い

### 1.3 楽天証券 マーケットスピード II
- **レイアウト思想**: 銘柄連携 + 2分割表示
- **特徴**:
  - 板・チャート・歩み値が「銘柄連携」で連動（1ヶ所変えると全部追随）
  - 「2分割表示」機能で2銘柄を並列比較しやすい
  - 板は気配上位8本〜10本まで表示可能
- **学べる点**: 「連携グループ」という概念。複数パネル間の銘柄同期はUX上強力

### 1.4 松井証券 ネットストック・ハイスピード
- **レイアウト思想**: 銘柄連動機能を持つコンパクトウィンドウ
- **特徴**:
  - 1ウィンドウで板・チャート・歩み値を切替表示
  - 銘柄連動グループで複数ウィンドウを同時切替
- **学べる点**: 軽量・コンパクト指向。常駐ツールとしての省スペース設計

### 1.5 マネックス証券 マルチボード500 ★最も参考になる
- **レイアウト思想**: 1シートで多銘柄板を一覧
- **特徴**:
  - **1シートで10銘柄の板情報を表示**
  - 「気配10」ボード = 多銘柄をコンパクトに並列表示
  - **板パーツが3本・5本・8本から選択可能**で、用途に応じて密度調整
  - 各セルが小さく、横並びで一望できる
- **学べる点**:
  - 「多銘柄を一度に視野に入れる」という板監視の本質ニーズに最も忠実
  - 板の深さを可変にすることで、スクリーニング用途と発注前確認用途を両立

## 2. 参考にすべきUI設計

### 2.1 採用すべき設計パターン

| 設計要素 | 採用元 | 採用理由 |
|---|---|---|
| 多銘柄並列表示 | マネックス マルチボード500 | 板監視の核心。1画面で10〜50銘柄を視野に入れる |
| 板の深さ可変（3/5/8本） | マネックス マルチボード500 | 用途別に密度を切り替えられる |
| 銘柄連携（複数パネル同期） | 楽天 マケスピII | 詳細確認時に複数パネルを一括追従 |
| タブによる銘柄グループ切替 | SBI HYPER SBI 2 系のタブUI | 50銘柄を5×10で分割し認知負荷を下げる |
| 売気配=赤・買気配=緑の色分け | 業界共通 | 色覚的に瞬時に売買方向を判別 |
| フラッシュ表示（更新検知） | 各社共通 | 約定/気配変化を視覚的に通知 |

### 2.2 避けるべき設計

- **expand=True による自動伸縮レイアウト**
  - 過去の v1 で採用→ウィンドウサイズ変更や銘柄追加でレイアウトが崩壊
  - 教訓: **固定幅（140px）・固定行高（13px）で expand=False を徹底**
- **過度なテンプレート押しつけ（旧 HYPER SBI 系の固定レイアウト）**
  - レイアウト変更のたびに再学習が必要になる
- **1銘柄=1ウィンドウ（kabuステーション素の状態）**
  - 50銘柄監視には不向き。タイル/タブで集約必須

### 2.3 当システム v2 で採用済みの設計

- 上段5タブ × 下段5タブ、各タブ5銘柄 = **計50銘柄**
- 板深さ: **売5段 + 買5段**（マネックスのパターンを参考）
- セル: 幅140px × 行高13px のコンパクト固定
- 色: 売=#330000/#ff6666、買=#003300/#66ff66
- フラッシュ: 売=ピンク #ffb6c1、買=水色 #add8e6
