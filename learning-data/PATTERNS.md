# 📈 地合い判断（全パターンの前提条件）

## 地合いを読む5つの指標

| 指標 | 確認内容 |
|------|---------|
| **日経平均** | 上昇 / 下落 / 横ばい の方向 |
| **TOPIX** | 日経と同方向か（確認用） |
| **騰落レシオ** | 市場全体の強弱（上昇銘柄÷下落銘柄） |
| **市場全体の出来高** | 盛り上がっているか（出来高増 = 本物の動き） |
| **先物の動き** | 翌日・場中の方向性を先読み |

## 地合いとペナントブレイク成功率の関係

```
良い地合い（日経↑ 出来高増 先物↑）
  → ペナントブレイク成功率 UP
  → 積極的にエントリー

悪い地合い（日経↓ 出来高減 先物↓）
  → ペナントブレイク成功率 DOWN
  → アラートが出ても慎重に / 見送りも選択肢

横ばい地合い（日経→ 方向感なし）
  → 銘柄選択が重要
  → 強い個別材料がある銘柄のみ狙う
```

## 実績との対応

| 日付 | 地合い | 検知数 | 成果 |
|------|--------|--------|------|
| 2026-05-22 | 強い上昇（防衛・グロース全面高） | 28件 | ストップ高3銘柄 |
| 2026-05-23 | ー | ー | ー |
| 2026-05-25 | **日経高値更新（AI・半導体全面高）** | 29件 | ストップ高1銘柄・+10%続出 |

**→ 地合いが強い日ほどペナントアラートの質が上がる**

## 毎朝の地合い確認ルーティン（maromaro実践）

```
【開場前】
  先物の値動きをチェック
  → 今日の方向感を掴む（↑強い / ↓弱い / →迷い）

【場中】
  先物チャートを出しながら監視
  → 個別のペナントアラートが出た時に
    先物も上向き → 積極エントリー
    先物が失速  → 慎重に / 見送りも選択肢
```

## 「一日上昇型」と「寄り天型」の見分け

```
一日上昇型（5/22・5/25のパターン）
  前場からアラート多発 → 後場もアラート続く
  先物が一日通して上向き
  → 乗り続けるのが正解

寄り天型
  前場にアラート出て急騰 → 後場失速・逆行
  先物が前場ピークに失速
  → 前場でさっさと利確が正解
```

**「地合いは開いてみないとわからない」（maromaro語録）**  
先物でヒントを掴みつつ、実際の動きで判断する。

### 板で読む「下落の底」（13:43の板）
- スクショ: `down-patterns/20260523-003-5803-fujikura-board.png`
```
OVER（売り板）:  1,802,500株
UNDER（買い板）: 4,210,100株  ← 2.3倍！ = UNDER優勢に逆転
```
- **UNDER > OVER に逆転 = 下落が止まって反転のシグナル**
- **4,820円に16,000株の買い板 = 下値目標・支持ライン**
- **チャートが下落中でも UNDER 優勢なら底が近い**

### 板で読む下値目標の計算
```
下値の買い板で最大の株数がある価格 = 出来高の壁 = 下値目標
この例: 4,820円（16,000株）が下値目標
```

### 板 × チャートの組み合わせ判断
| チャート | 板（OVER/UNDER） | 判断 |
|---------|----------------|------|
| 下落中 | OVER優勢 | 下落継続 → 撤退 |
| 下落中 | **UNDER優勢** | **底打ち近し → 反転を待つ** |

## パターン6: 機関投資家のふるい落とし（見せ板 → 支持線崩壊 → 急落） 🆕

- **学習日:** 2026/05/23
- **スクショ:** `down-patterns/20260523-004-5803-fujikura-shakeout.png`
- **銘柄事例:** 5803 フジクラ（4,820円の見せ板が崩壊 → 4,793円まで急落）
- **ユーザーコメント:** 「大口が個人をふるい落とした。新陳代謝が起きた状態」

### メカニズム
```
① 大口が大量買い板（見せ板）を入れる
   → 個人「ここが支持線！まだ保有できる」

② 価格が近づいた瞬間に大口がキャンセル
   → 支持線が突然消える

③ 個人が一斉に損切り売り
   → 価格が急落（4,820円 → 4,793円）

④ 大口が安値で買い集める（新陳代謝完了）
   → 個人の損切り売りを全部拾う

⑤ その後リバウンド
```

### 見せ板の見分け方
- **大量の注文が1つの価格に集中している**（4,820円に16,000株）
- **周辺の板より極端に大きい**（前後が2,000〜8,000株なのに16,000株）
- **価格が近づく直前に消える**（ここが確認ポイント）

### 時系列で見た値動き（フジクラ 5803 2026-05-23）
```
13:43  板: 4,820円に16,000株の買い板あり（UNDER 4,210,100株）
            → 個人「4,820円が支持線！」と誤認
13:46  4,793円 → 見せ板が消えて急落（-32円）
       安値 4,791円（4,800円割れ！）
       MACD: 4.2 / シグナル: 28.3（さらに悪化）
       出来高: 1,207,200（損切り売りが大量に出た）
```

### 上がるパターン5との関係
- **上がるパターン5**（4,800円割れ→リバ）= ふるい落とし後の**反発**
- **このパターン6** = ふるい落とし自体（**急落の局面**）
- 同じメカニズムだが「今見ているのが下落中か反転後か」で判断が変わる

### 板監視での活用
- 大きな買い板（周辺の5倍以上）= 見せ板の疑い
- 価格が近づいた時に板が消えたら → 急落に備える
- **OVER/UNDER 逆転アラートとの組み合わせ**: UNDER が急減したら見せ板消滅の可能性

## パターン7: ストップ狩りの階段（大口のキリ番罠 → 逆指値連鎖崩落） 🆕

- **学習日:** 2026/05/23
- **スクショ:** `down-patterns/20260523-IHI-stop-hunting/`
- **銘柄事例:** 7013 IHI（2026-05-23 15:11〜15:25）
- **ユーザーコメント:** 「大口があえて個人の逆指値を刈り取るためにぶち破る」

### メカニズム（完全版）

```
① 大口: キリ番（3,100円）に巨大な買い板を置く（79,000株）
   → 周辺の4〜15倍の突出した注文
   → 「ここが鉄板の支持線！」と個人が確信

② 個人: そのキリ番の少し下に逆指値を仕込む
   （例: 3,100円の壁 → 3,090〜3,095円に逆指値）

③ 大口: 壁を「増やしながら」価格を近づけていく
   （76,600 → 77,600 → 79,000 と増加 → 個人の確信が深まる）
   大口は同時にOVER側から大量売りをぶつけて価格を下げている

④ 大口: 自ら79,000株の壁をぶち破る（一気に売りを通す）
   → 3,100円割れ瞬間に個人の逆指値が一斉発動

⑤ 損切り売りが連鎖 → 価格が急落
   → 大口はさらに安値で買い集める or 空売りで利益
```

### 板ピカピカ = ブレイク予告サイン

```
キリ番の攻防が激しくなると歩み値が高速点滅（ピカピカ）する
→ この状態がブレイク直前の警告
→ 板ピカピカ + キリ番に突出した大口 = 急落準備完了
```

### ストップ狩りの階段（IHI 実例）

```
15:11  3,122円  観察開始（3,100円に76,600株の壁出現）
15:16  3,108円  板ピカピカ開始・壁が79,000株に成長
15:19  3,086円  3,100円（79,000株）ぶち破り → -18円瞬間崩落
15:20  3,075円  3,080円（25,300株）ぶち破り → -11円
15:21  3,067円  3,070円（16,900株）ぶち破り → -8円 安値3,067
15:22〜         大口が3,067〜3,070で買い集め開始
15:25  3,067円  引け・ドーリで底確認

合計: 3,122 → 3,067 = -55円（14分間）
```

### 壁が縮小するほど「売りがはけてきた」サイン

```
3,100円: 79,000株 （最大 = 売りの始まり）
3,080円: 25,300株 （中間）
3,070円: 16,900株 （縮小 = 売りがはけてきた）
→ 壁が小さくなるにつれて、大口の在庫が減っている
```

### 大口が買い板を売りに使う（パターン6との違い）

| | パターン6（見せ板） | パターン7（ストップ狩り） |
|--|--|--|
| 買い板 | 偽物・価格が近づく前に消える | **本物・実際にぶち破る** |
| 個人の行動 | 「支持線と思って保有」 | **逆指値を仕込ませる** |
| 大口の狙い | 高値で売り抜け | **個人の損切りを使って下げを加速** |
| 板の変化 | 突然消える | **壁が増えながら近づき→貫通** |

### 底打ちのサイン（反転の合図）

```
① OVER が減り始める（大口の売りがはけてきた）
② UNDER が増え始める（大口が買い始めた）
③ 大きな売り壁が突然消える（売り切り完了）
④ 歩み値に大口の緑（買い）が現れる（27,100株 @ 3,069など）
⑤ 価格がドーリで止まる（売り買いが拮抗 = 底）

IHI実例:
  OVER: 2,821,800 → 2,723,800（減少）
  UNDER: 1,719,400 → 1,737,400（増加）
  3,074円の35,800株売り壁が消えた
  27,100株の大口買いが歩み値に出現
  3,067円でドーリ引け
```

- 3,067〜3,070の仕込み価格を上抜けたら本格回復サイン

### 🔻 空売り戦略（このパターンの実践活用）

**エントリー条件（全部揃ったら入る）**
```
① MAが全部下向き = ダウントレンド確認
② キリ番に突出した厚い買い板が出現（周辺の5倍以上）
③ 板がピカピカ（歩み値が高速点滅・攻防激しい）
④ その壁が破られた瞬間 → 空売りエントリー
```

**ポジション継続の根拠**
```
次のキリ番にまた厚い板 → また破られた = 継続保有
また次のキリ番に厚い板 → また破られた = 継続保有
「破られ続ける間は下げ続ける」
```

**手仕舞い（買い戻し）のサイン**
```
① 厚い買い板が出た → 今度は破られない
② OVER が減り始める
③ UNDER が増え始める
④ 歩み値に大口の緑（買い）が出現
⑤ 大きな売り壁が消える
→ 複数揃ったら買い戻し
```

**IHI実例での空売り損益イメージ**
```
エントリー: 3,100円破られた瞬間（3,086円付近）
手仕舞い:   底打ちサイン確認（3,067〜3,070円）
利益:       約20〜30円/株
```

### 核心ルール（maromaro語録）

> 「移動平均線が全部下向きのダウントレンド中、厚い買い板が出てもそれを突き破って下げる。
> また厚い板が出てもまた突き破って下げる。
> この繰り返しが続く間は下げ続ける。
> 厚い買い板が出て、今度は破られなくなった時が本当の底。」

```
【下げ継続サイン】
MA全下向き + 厚い買い板 → 突き破られる = まだ下
               厚い買い板 → 突き破られる = まだ下
               厚い買い板 → 突き破られる = まだ下

【底確認サイン】
               厚い買い板 → 破られない  = 底
```

### 板監視での活用

- キリ番（3,000 / 3,050 / 3,100等）に突出した買い板 = 罠の可能性
- 周辺の5倍以上の注文 + OVER優勢 → ストップ狩り警戒
- 板ピカピカになったら → ブレイク直前
- 壁が増えながら価格が近づく → ブレイク確度が高い
- **「厚い買い板 = 安心」ではなく「厚い買い板 = 罠かもしれない」**

  - 必要データ: キリ番判定、板の突出注文検出、OVER/UNDER比率変化
