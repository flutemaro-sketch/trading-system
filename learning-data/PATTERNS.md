## パターン5: キリ番割れ → ふるい落とし → リバウンド 🆕

- **学習日:** 2026/05/23
- **スクショ:** `up-patterns/20260523-002-5803-fujikura.png`
- **銘柄事例:** 5803 フジクラ（2026-05-23 14:06 4,876円 +8.33%）
- **ユーザーコメント:** 「4800割れからのリバだね」
- **条件:**
  - 前場または序盤に大きく上昇している銘柄
  - 後場にキリの良い価格（xxx00 / xxx50 等）を**一時的に割り込む**
  - 割り込んだ直後に急反発（大陽線または連続陽線）
  - 下値が切り上がり始める（ペナントの下辺形成）
  - VWAP を上抜けで確認シグナル
- **なぜ上がるか:**
  - キリ番（4,800円）に集中していた損切り注文が全て刈り取られる
  - 売りが出尽くすと需給が逆転 → 急反発
  - 「ふるい落とし（shake out）」完了 = 次の上昇の起点
- **エントリーポイント:**
  - キリ番割れの安値から反発した最初の陽線確定後
  - または VWAP 回復・上抜けの確認後
- **注意点:**
  - キリ番を割り込んでも反発しない（さらに下落する）ケースもある
  - 出来高が増えているかどうかで判断（出来高増 = 本物のふるい落とし）
- **検知ロジック:** 未実装
  - 必要データ: キリ番判定、VWAP、直近の高値比較
- **パターン4との組み合わせ:** ふるい落とし後にペナント + VWAP 重なりが形成されたらさらに強い

### 【重要】パターン5 と パターン6 は表と裏の関係

```
パターン6（機関のふるい落とし）
    ↓ 個人が手放す（損切り）
    ↓ 機関が安値で買い集める
パターン5（4800割れからのリバ）← 完了の合図

= ふるい落とし完了 → 即リバウンド
```

**フジクラ 5803 2026-05-23 の完全な流れ**
```
13:06  天井（MACD デッドクロス）
13:43  見せ板 4,820円 16,000株 → 個人が「支持線」と誤認
13:46  見せ板消滅 → 4,793円急落（個人の損切りを刈る）← パターン6
       ↓ ふるい落とし完了・機関が買い集め
14:06  4,800円割れからリバウンド開始 ← パターン5の始まり
14:14  4,906円まで +113円急反発 🚀
```

**エントリー戦略（この2パターンを活用）**
- 大きな見せ板が消えて急落 → 慌てて売らない
- 4,800円（キリ番）を割り込んだ直後 = ふるい落とし完了の合図
- その足が陽線で終わったら → リバウンドエントリーのチャンス

### 時系列で見た値動き（フジクラ 5803 2026-05-23）
```
14:06  4,876円 → 水平抵抗線（4,877円）に到達
14:14  4,906円 → 水平抵抗線を突破、次のVWAP（4,908円）へ接近
出来高: 972,800（ブレイク時に急増 → 本物の上抜け確認）
```
- スクショ: `up-patterns/20260523-003-5803-fujikura-breakout.png`
- **水平線と斜め線が「×」交差した点を上抜け = 上昇加速のサイン**
- **ブレイク確認条件**: 抵抗線を上抜けた足の出来高が直前より増加していること

## パターン5: 後場 キリ番天井 + MACDデッドクロス → 下落 🆕

- **学習日:** 2026/05/23
- **スクショ:** `down-patterns/20260523-001-5803-fujikura-top.png`
- **銘柄事例:** 5803 フジクラ（2026-05-23 13:06 4,965円 +10.31%）
- **条件（全て重なると強い下落シグナル）:**
  - 前場で大きく上昇（+10%超）している
  - **キリ番（5,000円・10,000円等）手前で失速・跳ね返される**
  - **MACD がシグナルを下抜け（デッドクロス）or 直前**
  - MACDヒストグラムが赤に転換し始め（上昇モメンタム喪失）
  - 現在値が VWAP と同水準 or VWAP を下回り始め
  - 移動平均は上向きでも、勢いが落ちているのが重要
- **下落加速の引き金:**
  - VWAP 割れ → 損切り注文が集中して下落が加速
  - キリ番（5,000円）に何度も跳ね返されると売り圧力が強まる
- **エグジットの判断（保有中の場合）:**
  - MACD デッドクロス確認 → 利確検討
  - VWAP 割れ確認 → 即利確 or 損切り
- **検知ロジック:** 未実装
  - 必要データ: MACD・シグナル・VWAP・キリ番判定
- **注意点:**
  - 移動平均が全部上向きでも MACD で天井がわかることがある
  - 後場に入った直後（12:30〜13:00）が特に注意タイミング

### 時系列で見た実際の値動き（フジクラ 5803 2026-05-23）
```
13:06  4,965円  MACD57.5 / シグナル57.6  ← デッドクロス直前（警告）
13:42  4,825円  MACD13.4 / シグナル34.3  ← デッドクロス完了・VWAP割れ（-140円）
       出来高: 529,800 → 1,075,800（約2倍）← 大量売り確認
```
- スクショ: `down-patterns/20260523-002-5803-fujikura-fall.png`

### 下落フロー（完全版）
```
STEP 1: 前場 +10%超の急騰（始値4,501円 → 最高値5,000円手前）
STEP 2: 13:06 キリ番（5,000円）に跳ね返され、MACDがデッドクロス直前
STEP 3: 短期MA が中期MA を下抜け（デッドクロス確認）
STEP 4: VWAP（4,826円）を割り込む → 損切り売りが加速
STEP 5: 13:42 4,825円まで下落（-140円 / 36分間）
STEP 6: 出来高急増（売り出尽くし → その後リバの可能性も）
```

### 利確・損切りの最適タイミング
| タイミング | 根拠 |
|-----------|------|
| 13:06 MACD デッドクロス直前 | 最も早い警告 → 早めに一部利確 |
| 短期MA が中期MA を下抜け | デッドクロス確定 → 残りを利確 |
| VWAP 割れ（4,826円） | 下落加速の引き金 → 即撤退 |

### 板で読む「下落の底」（13:43の板）
- スクショ: `down-patterns/20260523-003-5803-fujikura-board.png`
```
OVER（売り板）:  1,802,500株
UNDER（買い板）: 4,210,100株  ← 2.3倍！ = UNDER優勢に逆転
```
- **UNDER > OVER に逆転 = 下落が止まって反転のシグナル**
- **4,820円に16,000株の買い板 = 下値目標・支持ライン**
- **チャートが下落中でも UNDER 優勢なら底が近い**

### 板で読む下値目標の計算
```
下値の買い板で最大の株数がある価格 = 出来高の壁 = 下値目標
この例: 4,820円（16,000株）が下値目標
```

### 板 × チャートの組み合わせ判断
| チャート | 板（OVER/UNDER） | 判断 |
|---------|----------------|------|
| 下落中 | OVER優勢 | 下落継続 → 撤退 |
| 下落中 | **UNDER優勢** | **底打ち近し → 反転を待つ** |
| 上昇中 | UNDER優勢 | 上昇継続 → 保有 |
| 上昇中 | OVER優勢 | 天井近し → 利確準備 |

## パターン6: 機関投資家のふるい落とし（見せ板 → 支持線崩壊 → 急落） 🆕

- **学習日:** 2026/05/23
- **スクショ:** `down-patterns/20260523-004-5803-fujikura-shakeout.png`
- **銘柄事例:** 5803 フジクラ（4,820円の見せ板が崩壊 → 4,793円まで急落）
- **ユーザーコメント:** 「大口が個人をふるい落とした。新陳代謝が起きた状態」

### メカニズム
```
① 大口が大量買い板（見せ板）を入れる
   → 個人「ここが支持線！まだ保有できる」

② 価格が近づいた瞬間に大口がキャンセル
   → 支持線が突然消える

③ 個人が一斉に損切り売り
   → 価格が急落（4,820円 → 4,793円）

④ 大口が安値で買い集める（新陳代謝完了）
   → 個人の損切り売りを全部拾う

⑤ その後リバウンド
```

### 見せ板の見分け方
- **大量の注文が1つの価格に集中している**（4,820円に16,000株）
- **周辺の板より極端に大きい**（前後が2,000〜8,000株なのに16,000株）
- **価格が近づく直前に消える**（ここが確認ポイント）

### 時系列で見た値動き（フジクラ 5803 2026-05-23）
```
13:43  板: 4,820円に16,000株の買い板あり（UNDER 4,210,100株）
            → 個人「4,820円が支持線！」と誤認
13:46  4,793円 → 見せ板が消えて急落（-32円）
       安値 4,791円（4,800円割れ！）
       MACD: 4.2 / シグナル: 28.3（さらに悪化）
       出来高: 1,207,200（損切り売りが大量に出た）
```

### 上がるパターン5との関係
- **上がるパターン5**（4,800円割れ→リバ）= ふるい落とし後の**反発**
- **このパターン6** = ふるい落とし自体（**急落の局面**）
- 同じメカニズムだが「今見ているのが下落中か反転後か」で判断が変わる

### 板監視での活用
- 大きな買い板（周辺の5倍以上）= 見せ板の疑い
- 価格が近づいた時に板が消えたら → 急落に備える
- **OVER/UNDER 逆転アラートとの組み合わせ**: UNDER が急減したら見せ板消滅の可能性
