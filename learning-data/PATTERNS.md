# パターン学習データベース

板監視システムで検知するパターンの学習記録。
スクショ + 条件 + 検知ロジックを蓄積していく。

**学習フロー:**
1. スクショを `up-patterns/` または `down-patterns/` に保存
2. このファイルにパターンを追記
3. 条件が固まったら検知ロジックをコードに実装

# ▲ 上がるパターン

## パターン1: ペナント上抜け ✅ **実装済み**

- **学習日:** 2026/05/18〜
- **スクショ:** `up-patterns/` 参照
- **実績:** 2026-05-22 に28件検知・3銘柄ストップ高達成
- **条件:**
  - 5分足でペナント形成（上値切り下がり + 下値切り上がり で収束）
  - 高値トレンドラインを上抜け
  - 収束率 35% 以上
  - R² ≥ 0.25（線形性あり）
  - 最低8本・最大50本のローソク足使用
- **検知ロジック:** `board_monitor/monitors/pennant_detector.py`
- **複数回発報の意味:**
  - 2回発報 → 上昇継続の強いシグナル
  - 3回発報 → ストップ高の可能性が高い（2026-05-22実績あり）
- **注意点:**
  - 一時的な上抜け後に戻ることもある（フジクラ事例 2026-05-22）
  - 出来高の確認も重要

# 📁 フォルダ構成

```
learning-data/
├── PATTERNS.md          ← このファイル（パターン記録）
├── up-patterns/         ← 上がるパターンのスクショ保存
│   └── YYYYMMDD-NNN.png
└── down-patterns/       ← 下がるパターンのスクショ保存
    └── YYYYMMDD-NNN.png
```

**スクショのファイル命名規則:**
```
up-patterns/20260525-001.png   （日付-連番）
down-patterns/20260525-001.png
```
