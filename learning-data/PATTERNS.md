# パターン学習データベース

板監視システムで検知するパターンの学習記録。
スクショ + 条件 + 検知ロジックを蓄積していく。

**学習フロー:**
1. スクショを `up-patterns/` または `down-patterns/` に保存
2. このファイルにパターンを追記
3. 条件が固まったら検知ロジックをコードに実装

## パターン1: ペナント上抜け ✅ **実装済み**

- **学習日:** 2026/05/18〜
- **スクショ:** `up-patterns/` 参照
- **実績:** 2026-05-22 に28件検知・3銘柄ストップ高達成
- **条件:**
  - 5分足でペナント形成（上値切り下がり + 下値切り上がり で収束）
  - 高値トレンドラインを上抜け
  - 収束率 35% 以上
  - R² ≥ 0.25（線形性あり）
  - 最低8本・最大50本のローソク足使用
- **検知ロジック:** `board_monitor/monitors/pennant_detector.py`
- **複数回発報の意味:**
  - 2回発報 → 上昇継続の強いシグナル
  - 3回発報 → ストップ高の可能性が高い（2026-05-22実績あり）
- **注意点:**
  - 一時的な上抜け後に戻ることもある（フジクラ事例 2026-05-22）
  - 出来高の確認も重要

## パターン5: キリ番割れ → ふるい落とし → リバウンド 🆕

- **学習日:** 2026/05/23
- **スクショ:** `up-patterns/20260523-002-5803-fujikura.png`
- **銘柄事例:** 5803 フジクラ（2026-05-23 14:06 4,876円 +8.33%）
- **ユーザーコメント:** 「4800割れからのリバだね」
- **条件:**
  - 前場または序盤に大きく上昇している銘柄
  - 後場にキリの良い価格（xxx00 / xxx50 等）を**一時的に割り込む**
  - 割り込んだ直後に急反発（大陽線または連続陽線）
  - 下値が切り上がり始める（ペナントの下辺形成）
  - VWAP を上抜けで確認シグナル
- **なぜ上がるか:**
  - キリ番（4,800円）に集中していた損切り注文が全て刈り取られる
  - 売りが出尽くすと需給が逆転 → 急反発
  - 「ふるい落とし（shake out）」完了 = 次の上昇の起点
- **エントリーポイント:**
  - キリ番割れの安値から反発した最初の陽線確定後
  - または VWAP 回復・上抜けの確認後
- **注意点:**
  - キリ番を割り込んでも反発しない（さらに下落する）ケースもある
  - 出来高が増えているかどうかで判断（出来高増 = 本物のふるい落とし）
- **検知ロジック:** 未実装
  - 必要データ: キリ番判定、VWAP、直近の高値比較
- **パターン4との組み合わせ:** ふるい落とし後にペナント + VWAP 重なりが形成されたらさらに強い

### 時系列で見た値動き（フジクラ 5803 2026-05-23）
```
14:06  4,876円 → 水平抵抗線（4,877円）に到達
14:14  4,906円 → 水平抵抗線を突破、次のVWAP（4,908円）へ接近
出来高: 972,800（ブレイク時に急増 → 本物の上抜け確認）
```
- スクショ: `up-patterns/20260523-003-5803-fujikura-breakout.png`
- **水平線と斜め線が「×」交差した点を上抜け = 上昇加速のサイン**
- **ブレイク確認条件**: 抵抗線を上抜けた足の出来高が直前より増加していること

## パターン4: ペナント上抜け + VWAP重なり + ゴールデンクロス 🆕

- **学習日:** 2026/05/23
- **スクショ:** `up-patterns/20260523-001-6920-lazertech.png`
- **銘柄事例:** 6920 レーザーテック（2026-05-23 13:53 38,220円）
- **条件:**
  - ペナント形成（下値切り上がり + 上値フラット or 切り下がり）
  - **上値抵抗ラインが VWAP と重なっている** ← 重要追加条件
  - ゴールデンクロス（短期MA > 長期MA）が発生している
  - VWAP を上抜けした瞬間がエントリーポイント
- **なぜ強いか:**
  - ペナントのブレイクポイント = VWAP という2つの抵抗が同時に突破される
  - ゴールデンクロスで上昇モメンタムが確認済み
  - 3条件が重なると上昇の確度が高い
- **パターン1（ペナント上抜け）との違い:**
  - パターン1: ペナント単体
  - パターン4: ペナント + VWAP + ゴールデンクロス の**3条件一致** → より信頼度高い
- **検知ロジック:** 未実装（VWAP・MA のデータ取得が必要）
- **実装優先度:** 高（VWAPデータがあれば比較的実装しやすい）
