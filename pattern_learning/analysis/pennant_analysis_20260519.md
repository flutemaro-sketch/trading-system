## パターンの本質（2日間の学習まとめ）

### 検知すべきパターン（3種類）

#### ① 上昇三角形（最多・主力）
```
上値: ほぼフラット（slope ≈ 0、わずかに負もOK）
下値: 切り上がり（slope > 0）
→ 収束して上値フラットをブレイク
```

#### ② 対称ペナント
```
上値: 切り下がり（slope < 0）
下値: 切り上がり（slope > 0）
→ 両側から収束してブレイク
```

#### ③ 上昇ウェッジ（スクエニ型）
```
上値: 切り上がり（slope > 0、ただし緩やか）
下値: 切り上がり（slope > 0、より急）
→ 両線とも上昇するが下値の傾きが大きく収束
```

### 検知不要なパターン
- **下駄GTA（日足）→ 朝から一日中上昇**: ペナントは作らない、前日検討システムの領域

## アルゴリズム修正方針

### 現在の問題点

if h_slope >= 0:
    return None  # ← 上昇三角形・上昇ウェッジを全部弾く！

### 修正後の判定ロジック
```python
# 上値ラインは「下値より傾きが小さい」ことを確認

    return None  # 収束していない（平行 or 開いている）

# 下値は必ず切り上がり

    return None
```
